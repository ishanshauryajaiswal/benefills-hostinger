## Analytics
- **Google Analytics 4 (GA4)**:
    - [How to add Google Analytics to Hostinger Website Builder](https://support.hostinger.com/en/articles/6438311-website-builder-how-to-add-google-analytics)
    - **Process**: Get Measurement ID (G-XXXXXXXX) from GA4 -> Edit Website -> Website Settings -> Integrations -> Google Analytics -> Paste ID -> Save -> Update Website.
    - **Timeline**: Data takes 24-48 hours to fully populate, but Realtime view works quickly.
